# Dashboard de Anúncios de Carros

Este projeto é um aplicativo web desenvolvido com Streamlit para visualizar dados de anúncios de venda de carros.

## Funcionalidades

- Exibição inicial dos dados
- Histograma da quilometragem dos veículos
- Gráfico de dispersão entre quilometragem e preço
- Aplicativo publicado na web usando Render

## Tecnologias usadas

- Python
- Pandas
- Plotly Express
- Streamlit
- Render
